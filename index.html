🖥️ SURVIE WINDOWS POUR LIONEL

Objectif : retrouver rapidement les manipulations les plus utiles du quotidien.

🏆 Les 5 raccourcis indispensables

Ctrl+C = Copier
Ctrl+V = Coller
Ctrl+X = Couper
Ctrl+A = Tout sélectionner
Ctrl+Z = Annuler

💡 Astuces à retenir

  • Un fichier HTML doit finir par .html
  • Un fichier texte classique finit par .txt
  • Si tu ne trouves plus une fenêtre : Alt + Tab
  • Si tout disparaît : Windows + D pour revenir au bureau
  • Avant de fermer un document important : Ctrl + S
